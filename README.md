# CHARGE RUN

## 概要
Unityを用いて制作した個人開発ゲームです。2ボタンで操作する2Dアクションゲーム。スペースキーで停止している間はチャージが溜まり、キーを離すとチャージ解放し一気に加速します。ジャンプキーで障害物を避けながらゴールを目指します。


## 操作方法
- ↑キー：ジャンプ
- スペースキー：停止（長押しでチャージ）
- スペースキーを離す：チャージ解放、加速
- escキー:PAUSE

## ゲームルール
- サボテン、トゲなどの障害物に触れるとゲームオーバー。
- トゲは近づくと左に動き出したり上下に動いたりしています。
- それ以外は当たると跳ね返ります。
- ゴールの旗に触れるとクリア。 

## 実装機能
- プレイヤー移動
- ジャンプ
- チャージ加速
- カメラ追従
- 障害物判定
- タイトル画面
- ゴール判定

## 更新内容
- アセットを使用してプレイヤー・地面・障害物・背景の見た目を調整
