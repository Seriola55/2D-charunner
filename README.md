# CHARGE RUN

## 概要
Unityを用いて制作した個人開発ゲームです。2ボタンで操作する2Dアクションゲーム。スペースキーで停止している間はチャージが溜まり、キーを離すとチャージ解放し一気に加速します。ジャンプキーで障害物を避けながらゴールを目指します。


## 操作方法
- ↑キー：ジャンプ
- スペースキー：停止（長押しでチャージ）
- スペースキーを離す：チャージ解放、加速
- escキー:PAUSE

## ゲームルール
- 赤い、茶色い障害物に触れるとゲームオーバー
- 茶色の障害物は近づくと左に動き出します
- 緑は触れても大丈夫ですが濃い緑は当たると跳ね返ります。
- 白いゴールラインに触れるとクリア 

## 実装機能
- プレイヤー移動
- ジャンプ
- チャージ加速
- カメラ追従
- 障害物判定
- タイトル画面
- ゴール判定

## 使用環境
- Unity 6
- C#
